# Clickify – E-Commerce Microservices Platform

## 🚀 Overview

This is a microservices-based e-commerce backend project built using Spring Boot and Spring Cloud.

---

## 🏗️ Services

* User Service
* Product Service
* Cart Service
* Order Service
* Payment Service
* Review Service

---

## 🔐 Features

* JWT Authentication
* API Gateway
* Service Discovery (Eureka)
* GraphQL Gateway

---
